👩🏽‍🚀 Projects

  • Kuchen Automat (Cake Machine)

    • A java program created solo as a university project on 2021. This software is created for the purpose of maintain/ administer a cake machine. It has the ability to manipulate cakes (CRUD) and at the same time automaticaly update the software GUI. Here I learned important aspects of a software such as unit tests, layer distribution, encapsulation, command line intefaces, thread safety, simple GUI, events, observers, and persistency.

  • Portfolio

    This portfolio is created using Three.js and with the purpose of listing all of my projects that I made/ contributed to. Here I took the first step towards 3 dimensional website programming and animation. Things that I learned includes camera, renderer, scene and css styling with animation. All credits to Fireship for teaching how to make 3d portfolio. Link to his youtube on github link below.

  • PackDieBadeHoseEin

    My first mobile iOS Application created with swift. The main purpose is to list all of the registered lake in Berlin and shows their names, coordinates, and exact location by using google map. The application has the ability to read JSON file and convert it into data. Here I learned about JSON File format and how to send and receive information to/from it. Another important lesson is how to build iOS simple GUI and how to add persistent items such as 'add to favorites'.

  • Tic Tac Toe

    My first java game created as a university project. The game features TCP networking which allows 2 players to play online against each other. The project uses test driven concept and object oriented programming as its base. I learned a lot of basic OOP rules, which was simple but can't be neglected, such as encapsulation, also known as access modifier, abstraction, inheritance, and polymorphism.
